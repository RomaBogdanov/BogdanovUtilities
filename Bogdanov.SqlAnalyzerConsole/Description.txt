Программа анализа данных в базах данных SQL (ред. от 15.10.21)

Программа расчитана на следующие типы СУБД:
MS SQL
Oracle
PostgreSQL

Функционал, который отрабатывает программа:
1. Поиск списка колонок в базе данных (реализовано для MS SQL, остальное в разработке);
2. Поиск списка колонок во всех базах данных сервера (реализовано для MS SQL, остальное 
   в разработке);
3. Поиск наличия значения или части значения в колонках базы данных (реализовано для 
   строковых значений MS SQL, остальное в разработке);
4. Поиск наличия значения или части значения в колонках всех баз данных сервера (
   в разработке);
5. Поиск наличия колонки в таблицах базы данных, выбранных баз данных или всех баз данных 
   СУБД (в разработке);
6. Поиск всего множества значений содержащегося в колонке конкретной таблицы или одинаково
   названной колонке выбранных(всех) таблиц БД или сервера БД (в разработке);
7. Поиск корреляций между колонками (в проекте);
8. Отслеживание изменений в базе данных, например, с целью отладки или реверс-инжиниринга
   приложений(в проекте);
