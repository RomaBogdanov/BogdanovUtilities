Задачи анализа БД:
1. Работа со строкой подключения к серверу:
1.1. Добавить строку подключения;
1.2. Удалить строку подключения;
1.3. Отредактировать строку подключения;
1.4. Сохранить список строк подключения к серверам;
1.5. Проверить работоспособность строки подключения;
1.6. Помощь в составлении строки подключения.
2. Выбор баз данных, которые принадлежат серверу:
2.1. Поиск всех входящих в сервер БД;
2.2. Выбор БД для анализа;
3. Выбор таблиц в БД:
3.1. Поиск всех таблиц в БД;
3.2. Выбор таблиц в БД;
3.3. Сохранение выбранных таблиц;
4. Поиск:
4.1. Поиск таблиц в БД по названиям;
4.2. Поиск колонок в БД по названиям;
4.3. Поиск значений в таблицах;
5. Бэкапирование:
5.1. Создание бэкапа таблицы;
5.2. Создание бэкапа БД.
6. Анализ:
6.1. Количество строк в таблице;
6.2. Объём БД.
